Data Science with R Learning Pathway

Inspired by the Open Source Data Science Master’s and Hadley Wickham’s advice on becoming a data scientist, this resource provides a pathway for learning Data Science with R. The foundation is the Johns Hopkins Data Science Specialization, supplemented with what someone should do next.

Statistical Knowledge

  • Brian Caffo’s Biostatitics Bootcamp: More mathematical foundation to the topics in Statistical Inference
  • Statistics in Medicine: More intuitive understanding of statistics plus introduction to non-parametric tests. After completing this class, I felt I had a much stronger understanding of concepts like p-values.
  • Designing, Running, and Analyzing Experiments: Practical approach covering parametric and non-parametric tests, along with the tests for the ANOVA assumption.

Programming Skills

Learning More R from Hadley

  • Advanced R: Quickly covers basics of R development before moving onto more advanced topics.
  • R Packages: The next step after abstracting your code with functions.

SQL

Most organizations store their data in a SQL database, and most non-RMDBS systems use a version of SQL as their query language.

  • Khan Academy
  • Mode’s SQL School
  • Stanford’s online course
  • dplyr Databases vignette

Functional Programming

  • The .*apply Family: Good overview of map-y functions in base-R.
